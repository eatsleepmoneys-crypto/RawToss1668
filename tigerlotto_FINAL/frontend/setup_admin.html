ตั้งค่า Super Admin Account
วิธีใช้:
1. ไป Railway → โปรเจค → Variables
2. เพิ่ม ADMIN_SETUP_KEY = รหัสลับที่คุณกำหนดเอง
3. กรอก Setup Key เดิมนั้นในช่องด้านบน
4. กรอกเบอร์โทรและรหัสผ่านสำหรับ Admin
5. กดปุ่มสร้าง — จะได้รับ token สำหรับ login

ล็อคอิน Admin: ไป admin.html
หน้าแอปหลัก: ไป index.html
